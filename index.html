Projetos em Destaque

De idéias a projetos, aqui estão aguns dos meu trabalhos mais recentes

  • Disparador de Mensagens do Whatsapp

    Este projeto foi desenvolvido em Python, utilizando a biblioteca Selenium para automatizar o envio de mensagens pelo WhatsApp. Destaca-se por sua arquitetura moderna, que integra um sistema de filas RabbitMQ, operando dentro de um container Docker. A implementação eficiente e escalável permite uma distribuição otimizada das mensagens, enquanto o armazenamento é gerenciado por um banco SQLite.

  • Disparador de E-mails

    Este projeto é uma solução eficiente para automatizar o envio de e-mails com anexos. Desenvolvido utilizando Python, o programa utiliza bibliotecas específicas para a manipulação de e-mails, como o smtplib para envio e o email.mime para anexar arquivos.

  • RPA

    Automação RPA em produção. Automatizando a interação com um sistema proprietário, a aplicação gera relatórios em Excel, extrai informações relevantes e as armazena em um banco de dados centralizado. Essa base de dados não apenas garante a consistência das informações, mas também serve como fonte para automações adicionais, como o disparo automático de mensagens e a geração de relatórios.
